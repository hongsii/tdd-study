# TDD 스터디 1주차

## TDD로 로또 통계 프로그램 개발

### 요구사항

* 기본 로또룰을 따름
* n개의 랜덤 로또 생성
    * 번호는 1-45 사이 숫자
    * 보너스 번호 존재
* 당첨번호 생성
* 로또 당첨 통계 출력 (등수별 당첨 횟수 및 획득 상금)

    | 등수 | 조건 | 상금 |
    |:---:|:---|:----|
    | 1등 | 6개 일치 | 100만원 |
    | 2등 | 5개 일치 + 보너스 일치 | 50만원 |
    | 3등 | 5개 일치 | 20만원 |
    | 4등 | 4개 일치 | 5만원 |
    | 5등 | 3개 일치 | 5천원 |
    | 꽝 | 나머지는 다 꽝 | 0 |
